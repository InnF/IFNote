# 基于概率的分类方法：朴素贝叶斯

## 基于贝叶斯决策理论的分类方法

 朴素贝叶斯
 + 优点：在数据较少的情况下仍然有效，可以处理多类别问题
 + 缺点：对于输入数据的准备方式较为敏感
 + 适用数据类型：标称型数据

 贝叶斯决策理论的核心思想是选择高概率对应的类别

## 朴素贝叶斯的一般过程

 + 收集数据：可以使用任何方法
 + 准备数据：需要数值型或布尔型数据
 + 分析数据：有大量特征时，绘制特征作用不大，此时使用直方图效果更好
 + 训练算法：计算不同的独立特征的条件概率
 + 测试算法：计算错误率
 + 使用算法：一个常见的朴素贝叶斯应用是文档分类。可以在任意的分类场景中使用朴素贝叶斯饭呢类其，不一定非要是文本
